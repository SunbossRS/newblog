unity！旧ubuntu桌面环境！

---

打开终端，输入命令：
```bash
sudo apt install unity-session
```
这时，终端让你确认yes or no，输入y。  
然后终端就在下载你的那个桌面环境了。  
下载好后，重启。  
进入你的账户，在出现的“小齿轮”图标选择你的桌面环境。
